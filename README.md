This project builds a grammar scoring engine for spoken audio samples as part of the SHL Intern Hiring Assessment.

## Methodology
- Extracted MFCC features from audio files using Librosa
- Trained a regression model to predict grammar scores
- Evaluated performance using RMSE
- Generated predictions for the test set and submitted via Kaggle

## Dataset
Dataset provided by SHL through Kaggle.

## Results
Public RMSE Score on Kaggle: **1.062**
